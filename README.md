# Advanced Warehouse Management System (Logistics Pro)

## Overview
A specialized backend system designed for large-scale warehouse operations. Unlike basic inventory tools, this project focuses on the full lifecycle of stock movement, including inbound logistics from suppliers and outbound tracking to customers.

## Key Features
- **Supplier Integration:** Managed database of external vendors and supply chain contacts.
- **Movement Tracking:** Detailed logging of 'IN' (Restocking) and 'OUT' (Sales/Distribution) transactions.
- **Real-time Stock Auditing:** Advanced SQL logic to calculate live stock levels based on historical movements.
- **Operational Analytics:** Reporting tools to identify supply chain bottlenecks and warehouse efficiency.

## Technical Skills Demonstrated
- **Transactional SQL:** High-performance queries for processing stock ins and outs.
- **Relational Constraints:** Implementation of `CHECK` constraints and `FOREIGN KEY` relationships for data accuracy.
- **Conditional Aggregation:** Using `CASE` statements within SQL to generate complex inventory summaries.
- **Logistics Logic:** Deep understanding of the supply chain workflow from procurement to delivery.

## Business Value
Optimizes warehouse space and reduces manual errors in stock counting. Essential for companies looking to digitize their logistics and supply chain management.
